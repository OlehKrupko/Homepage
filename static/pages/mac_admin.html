"$ " вводить НЕ НУЖНО. Создание загрузочного диска из образа Конвертируем *.iso-файл в *.img используя hdiutil. Вводим команду: $ hdiutil convert -format UDRW -o ~/path/to/target.img ~/path/to/ubuntu.iso Запускаем diskutil для получения перечня устройств. Запоминаем список устройств. $ diskutil list Вставляем флешку или другой носитель. Запускаем diskutil снова. Сравнивая с предыдущим списком определяем новое устройство и его номер (не раздела, а устройство) вида "/dev/diskN". $ diskutil list Запускаем, где N — это номер который мы узнали в предыдущем пункте. $ diskutil unmountDisk /dev/diskN Запускаем (обратите внимание на "r" в "rdiskN"), заменяя необходимое своимизначениями, где "/path/to/downloaded.img" — адрес образа диска, а "N" в "rdiskN" — номер пункта 5. $ sudo dd if=/path/to/downloaded.img of=/dev/rdiskN bs=1m Запускаем команду, где "N" — номер из пункта 5. Извлекаем носитель по окончании выполнения команды. $ diskutil eject /dev/diskN Готово! Носитель готов к использованию! Пустая иконка в Dock Добавить пустую иконку в левую часть (в раздел приложений). $ defaults write com.apple.dock persistent-apps -array-add '{"tile-type"="spacer-tile";}'; killall Dock Добавить пустую иконку в правую часть (в раздел папок). $ defaults write com.apple.dock persistent-others -array-add '{tile-data={}; tile-type="spacer-tile";}'; killall Dock Локальные копии Time Machine Если вы откроете About This Mac, то в разделе Storage можете увидеть, сколько места занимает локальная резервная копия.  Для того, чтобы отключить создание локальных бэкапов OS X, нужно выполнить в терминале следующую команду (потребуется ввод пароля администратора): $ sudo tmutil disablelocal Смысл отключения локального бэкапа может быть полезен в том случае, если у вас не так много свободного места на жёстком диске, например, он пригодится владельцам MacBook Air. Для того, чтобы включить функцию обратно, понадобится следующая команда: $ sudo tmutil enablelocal Для справки: файлы локальной резервной копии хранятся в директории /.MobileBackups. Если вы просканируете диск при помощи DaisyDisk в режиме администратора, то сразу обнаружите внушительный объем этой папки. Возможно ускорение процесса создания резервных копий (колдовство с Терминалом после каждой перезагрузки) Установка ПО из любого источника Отключить защиту $ sudo spctl --master-disable Включить защиту $ sudo spctl --master-enable
