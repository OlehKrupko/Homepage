What to do with slow USB speed on Asus router:

The situation is quite simple as routers have USB 3.0 disabled be default.

Feel free to follow the steps below to enable full USB 3.0 speed:

  1. Left sidebar >> "General" >> "Network Map" (it is first, default page).
  2. Click USB icon above "USB 3.0" text.
  3. Right sidebar "External USB disk status" >> click "Information" tab >> "USB Mode" section.
  4. Open dropdown by clicking on "USB 2.0" text >> choose "USB 3.0" option by clicking on it.
  5. Click "Apply" in the end to save the changes. If you wish, you can reboot the router.
