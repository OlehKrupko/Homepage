New features Start+A - Action Center Start+C - Cortana listening mode (used for voice commands) Start+S - Search / Launch Cortana (Listening mode) Start+I - Open the Settings app Start+Tab - Open the Task View Start+Ctrl+D - Create new Virtual Desktop Start+Ctrl+F4 - Close Current Desktop Start+Ctrl+[<<]OR[>>] - Switch between Virtual Desktops Standard features Start - Open / Close the Start Menu Start+X - Open / Close Start Menu's content (same as using right-click on mouse) Start+[<<]OR[>>] - Snap Current Window to left or right Start+E - Launch File Explorer Start+L - Lock the Desktop Alt+[Print Screen] - Copy Screenshot to Clipboard Start+[Print Screen] - Save screenshot to ~/Pictures/Screenshots (Windows 8.1-10) Browser shortcuts Ctrl+T - Open New Tab Ctrl+D - Bookmark Page Ctrl+L - Highlight curent URL (useful for copying URL or entering new one) Ctrl+Tab - Switch to next tab open in browser Ctrl+Enter - Add .com to end of URL (for example: type 'google' then press Ctrl+Enter)
