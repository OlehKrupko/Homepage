Basic commands CMD+P - print CMD+Q - Quit CMD+W - Close Window CMD+C - Copy CMD+S - Save CMD+V - Paste CMD+O - Open Window CMD+E - Eject CMD+N - New CMD++ - Zoom In CMD+- - Zoom Out CMD+M - Minimize CMD+X - Cut CMD+Z - Undo CMD+A - Select All CMD+H - Hide CMD+F - Find CMD+G - Find Next Screenshots: CMD+Shift+3 - Take full screen picture CMD+Shift+4 - Take selected screenshot CMD+Shift+4+Space - Take screenshot of selected window Finder (go to folder): CMD+Shift+C - Computer ( / ) CMD+Shift+A - Applications ( /Applications/ ) CMD+Shift+H - Home ( ~ ) CMD+Shift+G - Open "Go to" menu
