Docker cheatsheet

Commands: Maintenance

stop and remove all containers: docker container stop $(docker container ls -aq); docker container rm $(docker container ls -aq) remove all image cache: docker image prune -a clear Docker data: docker container stop $(docker container ls -aq); docker container rm $(docker container ls -aq); docker image prune -a docker-compose up: docker-compose up docker-compose up (build from scratch, remove containers from previous deprecated versions of docker-compose, recreate containers even if their configuration and image haven't changed.): docker-compose up --build --remove-orphans --force-recreate
